## FAQ

### How to choose between CIF, MD or Data Vault?

  - CIF (Inmon)
  	- Easy to maintain
	  - Structured
	  - Easier data mining
	  - Timely to build
  - MD (Kimball)
	  - Start small, scale big
	  - Faster ROI
	  - Analytical Tools
	  - Low reusability
  - Data Vault (Lyndsted)
	  - Backend Data Warehouse
	  - Multiple sources; Full history; Incremental build
	  - Up-front work; Long-term payoff; Many joins
